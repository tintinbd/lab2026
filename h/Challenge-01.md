# Create a Github repository for python project

## Goal

You will need to create a Github repository which will comtain python application's code.

## Requirement

1. Choose a suitable name for Github account and python repository
2. Your Github account should have minimal login security 
3. Add this repository's 'tintinbd' account as a contributing member to your repository
4. Anyone can view your repository, but only yours and 'tintinbd' accounts can make changes
5. Direct commit to 'main' branch is not allowed. A PR from only 'develop' branch is required to commit any change to 'main' branch
6. At least one approval is needed when merging code to 'main' branch 
7. Direct commit to 'develop' branch is not allowed. A PR from another branch is required to commit changes to 'develop' branch
8. Creare a simple python file and readme file, and commit to a new brnach.
9. Create a PR and merge to develop branch
10. Create a PR for main branch

## Restriction 

1. No restriction on using CoPilot
2. Don't share your work with other students.
3. Please complete as much as possible without asking any further clarifying question.
